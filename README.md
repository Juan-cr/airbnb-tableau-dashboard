# Project Overview

This project explores Airbnb listing data through interactive Tableau visualizations to uncover pricing trends, revenue opportunities, and geographic distribution patterns.

The dashboard was created to transform raw Airbnb data into actionable business insights using interactive charts, maps, KPI metrics, and trend analysis.

This project demonstrates skills in:
- Data Visualization
- Dashboard Design
- Geographic Analysis
- Trend Analysis
- Business Intelligence
- Interactive Reporting

---

# Business Understanding

The purpose of this project was to analyze Airbnb listing performance and identify trends related to:
- Average pricing by bedroom count
- Revenue patterns throughout the year
- Geographic pricing distribution
- Listing concentration by bedroom size

The dashboard was designed to help understand how Airbnb pricing and revenue vary across different locations and property sizes.

This project also served as practical experience in building professional business dashboards using Tableau.

---

# Data Understanding

The dataset includes Airbnb listing information such as:

- Average pricing

# Key Insights

- Properties with more bedrooms generated higher average listing prices.
- One-bedroom listings represented the majority of available Airbnb properties.
- Revenue trends remained strong and relatively stable throughout the year.
- Certain zipcodes consistently showed higher average prices than others.
- Geographic visualizations revealed pricing concentration patterns across regions.
